# 📚 Flashcard Quiz App

A Flutter-based Flashcard Quiz Application developed as part of the **CodeAlpha App Development Internship**. Helps users create, manage, and review flashcards for effective self-study.

## ✨ Features

- Create, edit, and delete flashcards
- Search flashcards instantly
- Quiz Mode with Show / Hide Answer
- Previous / Next card navigation
- Local storage with Hive Database
- Light & Dark Theme support
- Clean Material 3 UI

---

## 🛠️ Built With

- **Flutter** — Cross-platform UI framework
- **Dart** — Programming language
- **Hive** — Lightweight local database
- **Material 3** — Modern UI design system

---

## 📦 Packages Used

| Package        | Purpose                       |
| -------------- | ----------------------------- |
| `hive`         | Local database                |
| `hive_flutter` | Hive integration with Flutter |
| `provider`     | Theme management              |

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio / VS Code
- Android Emulator or Physical Device

### Run Locally

```bash
git clone https://github.com/Wajid-Ali-130/CodeAlpha_FlashcardQuizApp.git
cd CodeAlpha_FlashcardQuizApp
flutter pub get
flutter run
```
